Class: FieldDate

FieldDate

new FieldDate(date, opt_validatoropt)

Class for a date input field.
Parameters:
Name Type Attributes Description
date string The initial date.
opt_validator function <optional>
 A function that is executed when a new date is selected. Its sole argument is the new date value. Its return value becomes the selected date, unless it is undefined, in which case the new date stands, or it is null, in which case the change is aborted.
